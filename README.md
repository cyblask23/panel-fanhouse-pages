# panel-fanhouse-pages

## 项目简介

`panel-fanhouse-pages` 是一个用于归档和发布独立 HTML 页面的仓库。本项目不针对任何特定域名或网站，仅作为页面文件的存储与展示空间。

仓库中收录的页面均为独立的静态 HTML 文件，适用于快速浏览、参考或离线使用场景。所有内容以原始文件形式保存，不涉及动态后端或数据库。

## 目录说明

```
.
├── pages/          # 存放所有 HTML 页面文件
├── assets/         # 页面引用的资源文件（CSS、JS、图片等）
├── index.html      # 仓库页面索引（可选）
└── README.md       # 本文件
```

- `pages/`：核心目录，每个 HTML 文件为一个独立页面，文件名通常反映页面主题或用途。
- `assets/`：存放页面所需的静态资源，按需组织子目录。
- `index.html`：可选的导航页，用于快速浏览仓库内所有页面。

## 页面归档说明

本仓库收录的页面满足以下条件：

- 所有页面均为纯前端静态 HTML 文件，不依赖外部服务或 API。
- 页面内容由贡献者自行整理或创作，不涉及商业用途或具体服务推广。
- 归档页面保持原始结构，不进行功能性修改或优化。
- 每个页面都尽可能声明其来源或归档日期（可在页面注释中查看）。

## 维护说明

- 欢迎提交 Pull Request 添加新的独立 HTML 页面。
- 提交前请确保页面不包含恶意脚本、追踪代码或违反法律的内容。
- 页面应尽量自包含，资源文件建议放入 `assets/` 目录。
- 如发现页面无法正常访问或包含不适宜内容，请提交 Issue 说明情况。
- 仓库维护者保留对收录页面进行修改或移除的权利。
